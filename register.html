¿Cómo se usa?How to use it

Es un synth, no un sampler — no carga audio: cada preset es un juego de números que el motor interpreta. It is a synth, not a sampler — it loads no audio: each preset is a set of numbers the engine interprets.

  1. Elegí un instrumento en el selector → ya queda aplicado al motor. Tocá nota para oírlo.Pick an instrument in the selector → it is applied to the engine right away. Hit note to hear it.
  2. Copy JSON → vuelca sus números a la caja de texto (modo edición).Copy JSON → dumps its numbers into the text box (edit mode).
  3. Editá los números, o pegá un JSON de afuera.Edit the numbers, or paste a JSON from outside.
  4. stop — que no quede ninguna nota sonando.stop — make sure no note is left sounding.
  5. Apply from box → aplica lo que esté en la caja al motor.Apply from box → applies whatever is in the box to the engine.
  6. nota → lo escuchás y verificás. Volvés al paso 2 las veces que quieras.note → listen and check. Go back to step 2 as many times as you like.
  7. Download .txt → lo baja como <nombre>.txt (el nombre sale del campo name).Download .txt → saves it as <name>.txt (the name comes from the name field).
  8. ↺ preset / ↺ factory — el botón dice lo que hará: con un preset activo, el primer clic deshace tus ediciones (vuelve al preset); el segundo va a fábrica.↺ preset / ↺ factory — the button says what it will do: with an active preset, the first click undoes your edits (back to the preset); the second goes to factory.
Mismo modelo que el motor: p = posición logarítmica de la nota en el rango (0 grave, 1 agudo); las cuatro respuestas se interpolan entre grave/medio/agudo y escalan amplitud, ruido de soplo, parciales agudos y tiempo de ataque. Las curvas son la forma del registro (cambian por instrumento); el timbre base es el carácter. Same model as the engine: p = the note's logarithmic position in the range (0 low, 1 high); the four responses interpolate between low/middle/high and scale amplitude, breath noise, upper partials and attack time. The curves are the register's shape (they change per instrument); the base timbre is its character.
